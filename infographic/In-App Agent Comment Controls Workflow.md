# In-App Agent Comment Controls Workflow

```text
X signal
  ↓
Structured capture fields
  ↓
Agent-ready task packet
  ↓
Concrete artifact / patch / memo
  ↓
Evidence gate
  ↓
Promote, iterate, or hold
```

## Why it matters
Visual feedback should happen where the product problem is visible: click a region, leave a natural-language edit request, and turn that into an agent patch with evidence.

## First measurable proof
Run one real VinClawLabs workflow through the prototype and record the before/after artifact plus decision outcome.
